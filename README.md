# gw2_BoonFX

# Boon bar and visualizer

Displays Stability, Quickness, Might, Alacrity, and Stability on a boon bar underneath the character

* Flame effects behind the boon bar when Quickness, Might, or Alacrity are applied. Flames grow to red when Might is over 20 stacks.

* Shifting Bricks effect behind the boon bar when Stabiltiy is applied.

* Glowing blue orb around Health when Aegis is applied.

Each component can be toggled separately

# Installation Guide

Simply extract the zip file inside the reffect folder, and the icons and packages folders will be updated with the Paragon package. Refresh packs in-game afterwards.
